# Third-Party Assets and Licensing

This document records the source and license of every external component
used to generate the SynLoc dataset, in compliance with reviewer R2-V
and the *Data and Code Availability* statement of the paper.

Only **outputs** generated by our pipeline (rendered RGB images, depth
maps, camera poses, intrinsics, covisibility metadata, train/val/test
splits, generation/training/evaluation scripts) are redistributed under
the licences declared in `LICENSE` (MIT, code) and `LICENSE-DATA`
(CC-BY-4.0, dataset). **No third-party engine binary, proprietary asset
or copyrighted source map is redistributed.**

---

## 1. Geographic Information / City Maps

| Component                       | Source                          | License           | Use in SynLoc                                         |
|---------------------------------|---------------------------------|-------------------|-------------------------------------------------------|
| Road network / building footprints | OpenStreetMap (OSM) contributors | ODbL 1.0          | Procedural city layout (block subdivision, road graph) |

We use OSM data **only as procedural input**; no OSM tile, vector dump
or derivative map is redistributed. Any downstream user wishing to
regenerate city layouts must download OSM data themselves under ODbL
and follow the attribution clause:
"© OpenStreetMap contributors, https://www.openstreetmap.org/copyright".

## 3. Textures and Materials

| Component                  | Source                            | License                | Redistributed?                  |
|----------------------------|------------------------------------|------------------------|----------------------------------|
| Building facade textures   | Authored by us / CC-0 photo sets   | CC-BY-4.0 / CC-0       | Yes (as part of dataset)         |

| Sky / HDRI                 | PolyHaven HDRIs (where applicable) | CC-0                   | Optional re-distribution         |

Every texture file shipped in the dataset release carries an SPDX
licence header (or accompanying `.license` sidecar). Textures we cannot
redistribute are referenced by stable URL only and must be obtained by
the user from the original vendor.

## 5. Code Dependencies

All Python / C++ dependencies and their licences are pinned in
`environment.yml` and `requirements.txt`. None are GPL or AGPL. The
SynLoc generation, training and evaluation code itself is released
under the MIT licence (see `LICENSE`).

## 6. Pre-trained Models Used for Comparison

Pre-trained checkpoints of LoFTR, RoMa and other matchers used for
evaluation retain the licences distributed by their original authors.
We do **not** redistribute these checkpoints.

## 7. Ethics

The SynLoc dataset is procedurally generated and contains no human
subjects, personally identifiable information or animal data. Real-data
evaluation is performed exclusively on the public MegaDepth-1500
benchmark under its original licence.
